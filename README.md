# qa-playwright-tests

A sample [Playwright](https://playwright.dev/) test suite.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18 or later

### Installation

Install dependencies:

```bash
npm install
```

Install Playwright browsers:

```bash
npx playwright install
```

### Running Tests

Run all tests:

```bash
npm test
```

View the HTML test report:

```bash
npm run test:report
```

## Project Structure

```
├── tests/
│   └── example.spec.ts   # Sample Playwright tests
├── playwright.config.ts  # Playwright configuration
└── package.json
```
